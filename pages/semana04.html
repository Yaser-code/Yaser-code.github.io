¿Qué aprendí?

JavaScript

lenguaje de programación interpretado, ligero y orientado a objetos, ampliamente utilizado para desarrollar sitios web dinámicos e interactivos.

Principales funciones:

  • Validación de formularios.
  • Animaciones.
  • Interacción con el usuario (eventos, clics, teclado, etc.).

Caracteristicas:

  • Lenguaje interpretado (no necesita compilarse).
  • Orientado a objetos y basado en prototipos.
  • Compatible con todos los navegadores modernos.
  • Muy utilizado junto a HTML y CSS para desarrollo web frontend.

Typescript

es un superset de JavaScript desarrollado por Microsoft. Esto significa que extiende a JavaScript agregándole características avanzadas, siendo la más destacada el tipado estático.

Caracteristicas:

  • Tipado estático: puedes declarar tipos (string, number, boolean, etc.).
  • Orientado a objetos con clases, interfaces y herencia más avanzadas.
  • Mejora la escalabilidad de proyectos grandes.
  • Compatible con librerías y frameworks como Angular, React y Vue.

DECLARACIÓN DE VARIABLES CON TIPOS EN TYPESCRIPT

Forma de declarar Uso
let nombre: string = "Ana"; Declara una variable tipo texto (cadena de caracteres)
let edad: number = 25; Declara una variable numérica
let esEstudiante: boolean = true; Declara una variable booleana (verdadero o falso)
let notas: number[] = [18, 17, 20]; Declara un arreglo de números

EJEMPLOS PRÁCTICOS EN TYPESCRIPT

Ejemplo Uso
function saludar(nombre: string): void {
  console.log("Hola " + nombre);
}		 Función que recibe un texto y no retorna valor (`void`)
let suma = (a: number, b: number): number => {
  return a + b;
};		 Función flecha que suma dos números y retorna el resultado
interface Persona {
  nombre: string;
  edad: number;
}

  edad: 22
};		 Interface para definir la estructura de un objeto

datos = 123;
