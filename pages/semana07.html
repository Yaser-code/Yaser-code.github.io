¿Qué aprendí?

Hooks en React

Los Hooks son funciones especiales introducidas en React que permiten usar el estado y otras características de React (como el ciclo de vida, el contexto, etc.) sin escribir clases.

Objetivo

Reutilizar lógica de componentes y trabajar con funcionalidades avanzadas en componentes funcionales.

Hooks Básicos

Hooks Uso
useState Para manejar estado local.
useEffect Para efectos secundarios (fetch, subscripciones, etc.).
useContext Para consumir un contexto sin usar el componente <Context.Consumer>

Hooks Avanzados

Hooks Uso
useRef Para referencias mutables o acceder al DOM.
useReducer Para manejar lógica de estado más compleja (similar a Redux).
useCallback Para memorizar funciones y evitar renders innecesarios.
useMemo Para memorizar valores calculados.

Ejemplo practico de useState


                    import React, { useState } from 'react';

                    function Contador() {
                        const [contador, setContador] = useState(0);

                        const incrementar = () => {
                            setContador(contador + 1);
                        };

                        return (
                            <div>
                            <h2>Contador: {contador}</h2>
                            <button onClick={incrementar}>Incrementar</button>
                            </div>
                        );
                    }

Hooks Personalizados en React

Los Hooks personalizados son funciones de JavaScript que utilizan uno o más Hooks de React en su interior (useState, useEffect, etc.) para encapsular y reutilizar lógica de componentes.

Regla principal

Siempre deben comenzar con use en su nombre (por convención y para que React los reconozca como hooks).

Por qué debemos usarlo:

  • Reutilizar lógica entre componentes sin repetir código.
  • Mejorar la organización y legibilidad del código.
  • Separar responsabilidades (principio de responsabilidad única).

Ejemplos de Hooks personalizados

Hooks de Ejemplo Función de Hook
useForm Manejo de formularios (inputs, validaciones)
useFetch Lógica de peticiones a APIs
useLocalStorage Leer y escribir valores en localStorage
useDebounce Retardar el cambio de un valor (útil en búsquedas)
useToggle Activar/desactivar booleanos (switch, modal, etc.)
usePrevious Obtener el valor anterior de una variable

Reglas para crear hooks personalizados

  • Deben empezar con use (por ejemplo, useLogin, useTimer, etc.).
  • Solo pueden usarse dentro de componentes o de otros hooks (nunca fuera del contexto de React).
  • Pueden usar otros hooks dentro (useEffect, useState, etc.).

Ejemplo práctico

Creando un hook useContador


                        import { useState } from 'react';
                            function useContador(inicial = 0) {
                            const [contador, setContador] = useState(inicial);

                            const incrementar = () => setContador(contador + 1);
                            const decrementar = () => setContador(contador - 1);
                            const resetear = () => setContador(inicial);

                            return { contador, incrementar, decrementar, resetear };
                        }

Usando el hook en un componente


                        function Contador() {
                        const { contador, incrementar, decrementar, resetear } = useContador(5);

                        return (
                            <div>
                            <h1>{contador}</h1>
                            <button onClick={incrementar}>+1</button>
                            <button onClick={decrementar}>-1</button>
                            <button onClick={resetear}>Reset</button>
                            </div>
                        );
                        }
