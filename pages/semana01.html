Fundamentos de las tecnologías web

Los fundamentos de las tecnologías web son los principios, herramientas y lenguajes básicos que permiten el desarrollo, funcionamiento y visualización de sitios y aplicaciones en Internet.

Soluciones web:

  • Sistema web:

    Conjunto de herramientas, programas y bases de datos que funcionan en la web para cumplir un objetivo específico, como gestionar ventas o usuarios.

  • Aplicación web:

    Programa interactivo que se ejecuta en un navegador y permite realizar tareas específicas.

  • Sitio web:

    Conjunto de páginas web relacionadas entre sí, accesibles desde una misma dirección.

  • Página web:

    Documento único de un sitio web, creado con HTML, que muestra contenido específico como texto, imágenes o formularios.

Tecnologías web:

  • Básicas:

    Tecnologías web básicas.

    • HTML: Lenguaje que estructura el contenido de una página web.
    • CSS: Lenguaje que da estilo al contenido (colores, tamaños, diseño).
    • JavaScript: Lenguaje que permite agregar interactividad y dinamismo.
    • HTTP/HTTPS: Protocolos que permiten la comunicación entre navegador y servidor.
    • URL: Dirección única que identifica un recurso en la web.
  • Lenguajes:

    Lenguajes web

    • Frontend: HTML, CSS, JavaScript - se ejecutan en el navegador.
    • Backend: PHP, Python, Java, Node.js - se ejecutan en el servidor.
    • SQL: Lenguaje para gestionar bases de datos.
  • Gráficos:

    Tecnologías gráficas en la web.

    • SVG: Formato de gráficos vectoriales escalables usados en la web.
    • Canvas (HTML5): Elemento que permite dibujar gráficos, animaciones o juegos.
    • WebGL: Tecnología para renderizar gráficos 3D dentro del navegador.
    • CSS3: Permite aplicar efectos visuales como sombras, transiciones y animaciones.

Funcionamiento de la web:

  • DNS(Sistema de Nombres de Dominio):

    Traduce nombres de dominio (como www.google.com) a direcciones IP (como 142.250.190.36) para que los navegadores puedan encontrar los servidores correctos.

  • Protocolo TCP/IP:

    Conjunto de reglas que permite que los datos viajen por Internet.

    • TCP (Transmission Control Protocol): Divide la información en paquetes y asegura que lleguen correctamente.
    • IP (Internet Protocol): Se encarga de direccionar los paquetes al destino correcto.
  • Protocolo HTTP(HyperText Transfer Protocol):

    Es el protocolo que permite la comunicación entre el navegador (cliente) y el servidor web.

    • Usa solicitudes (como GET o POST) para pedir o enviar información.
    • Cuando es seguro, se llama HTTPS, e incluye cifrado (SSL/TLS).

Roles en el desarrollo de Aplicaciones para Internet

  • Desarrollador frontend:

    Se encarga de la parte visual e interactiva de una web, lo que ve y usa el usuario.

  • Desarrollador backend

    Se ocupa de la lógica del servidor, bases de datos y seguridad. Maneja lo que no se ve, pero hace que todo funcione.

  • Desarrollador fullstack:

    Combina ambos roles: puede trabajar tanto en el frontend como en el backend. Tiene conocimientos integrales del desarrollo web completo.
