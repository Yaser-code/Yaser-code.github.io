Creación de Proyectos con Spring Boot

Spring Boot simplifica la creación de proyectos al ofrecer una estructura preconfigurada con dependencias listas para usar. Se utiliza principalmente con Spring Initializr.

rocket_launch ¿Qué hace Spring Boot?

  • Automatiza configuraciones comunes (como conexión a la base de datos).
  • Incluye un servidor embebido (como Tomcat).
  • Permite ejecutar aplicaciones fácilmente con main().

Persistencia de Datos con Spring Data JPA

Spring Data JPA permite trabajar con bases de datos utilizando objetos y clases Java, sin escribir SQL directamente. Usa interfaces que Spring implementa automáticamente.

database ¿Cómo se estructura?

  • Una clase entidad anotada con @Entity.
  • Un repositorio que extiende JpaRepository.
  • Spring se encarga del CRUD completo.

select_check_box Ejemplo de Entidad y Repositorio


                        @Entity
                        public class Usuario {
                            @Id
                            @GeneratedValue(strategy = GenerationType.IDENTITY)
                            private Long id;
                            private String nombre;
                            private String correo;
                            // getters y setters
                        }
                    

                        public interface UsuarioRepository extends JpaRepository<Usuario, Long> {
                            // Métodos como findAll(), findById(), save(), delete() ya están listos
                        }

check_circle Ventajas de Spring Data JPA

  1. Reduce el código repetitivo de SQL.
  2. Fácil integración con controladores REST.
  3. Permite consultas personalizadas con métodos simples.
