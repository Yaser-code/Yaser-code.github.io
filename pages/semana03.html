¿Qué aprendí?

Libreria Bootstrap

Bootstrap es un framework de CSS, JavaScript y HTML desarrollado por Twitter, que te permite crear sitios web responsivos y modernos rápidamente.

done_all Lo que ofrece:

  • Componentes preconstruidos: botones, formularios, navbars, alertas, modales, carouseles, etc.
  • Diseño responsivo fácil con clases
  • Clases utilitarias para margen, padding, colores, alineación, etc.
  • Estilo uniforme y profesional desde el inicio
  • Basado en HTML, CSS y JavaScript
  • Compatible con navegadores modernos y antiguos
  • Rápido para prototipar sitios web

close Desventajas:

  • Diseños repetitivos o genéricos
  • Limitado a su estilo base
  • Flexibilidad limitada en diseño
  • Aprendizaje de sus clases propias

Libreria Tailwind

Tailwind CSS es un framework de CSS de utilidad que te permite construir interfaces de usuario modernas y responsivas directamente desde tus clases HTML, sin tener que escribir tanto CSS personalizado.

done_all Lo que ofrece:

  • Clases de utilidad muy específicas para cada estilo
  • Estilizado directamente desde HTML
  • Sistema de diseño responsivo muy flexible con prefijos
  • Alto nivel de personalización
  • Diseño moderno, limpio y minimalista por defecto
  • Fomenta un diseño a medida, sin componentes preestablecidos

close Desventajas:

  • HTML muy cargado de clases
  • Curva de aprendizaje inicial
  • Sin componentes prehechos
  • Menos útil para prototipar rápidamente componentes completos

start Ejemplos practicos:

Estilo Código Ejemplo
Boton con Tailwind <p><button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">Hacer clic</button></p>
