Introducción a Flask: Servidor Web Básico

Flask es un microframework en Python que permite crear aplicaciones web de manera rápida y sencilla. Su estructura es ligera y flexible.

¿Cómo funciona Flask?

  • Define rutas (URLs).
  • Asocia funciones a esas rutas.
  • Devuelve respuestas al navegador.

Ejemplo básico de servidor Flask


                    from flask import Flask

                    app = Flask(__name__)

                    @app.route('/')
                    def inicio():
                        return 'Hola mundo con Flask'

                    @app.route('/saludo/<nombre>')
                    def saludar(nombre):
                        return f'Hola, {nombre}! Bienvenido a Flask.'

                    if __name__ == '__main__':
                        app.run(debug=True)

Este pequeño servidor tiene dos rutas: una para el inicio y otra para saludar usando un parámetro dinámico.

¿Por qué usar Flask?

  1. Es fácil de aprender para quienes ya dominan Python.
  2. Ideal para proyectos pequeños y medianos.
  3. Permite extender fácilmente con librerías como Jinja2, SQLAlchemy, entre otras.
