Servlets

Un Servlet es una clase Java que se ejecuta en un servidor web y responde a solicitudes HTTP (como cuando un usuario envía un formulario o accede a una URL).

star_rate_half ¿Qué puedes hacer con Servlets?

  • Procesar formularios.
  • Generar HTML dinámico.
  • Conectarte a una base de datos.
  • Controlar el flujo de navegación.

view_comfy_alt Componentes clave

  • HttpServlet: Clase base para los servlets HTTP.
  • doGet() y doPost(): Métodos para manejar peticiones GET y POST.
  • HttpServletRequest y HttpServletResponse: Objetos para leer datos enviados y generar respuestas.

select_check_box Ejemplo practico de Servlet básico


                        @WebServlet("/saludo")
                        public class SaludoServlet extends HttpServlet {
                            protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
                                response.setContentType("text/html");
                                PrintWriter out = response.getWriter();
                                out.println("<h1>Hola desde un Servlet en Java>/h1>");
                            }
                        }

Conexión a Bases de Datos con JDBC

JDBC (Java Database Connectivity) es una API que permite a las aplicaciones Java conectarse y trabajar con bases de datos relacionales como MySQL, PostgreSQL, Oracle, etc.

star_rate_half Pasos para usar JDBC

  • Cargar el driver JDBC.
  • Establecer conexión con la base de datos.
  • Crear y ejecutar consultas SQL.
  • Procesar los resultados.
  • Cerrar la conexión.

select_check_box Ejemplo práctico: Insertar un usuario en una tabla


                        import java.sql.*;

                        public class ConexionBD {
                            public static void main(String[] args) {
                                String url = "jdbc:mysql://localhost:3306/mi_base";
                                String user = "root";
                                String password = "123456";

                                try {
                                    // 1. Conectar a la base de datos
                                    Connection conexion = DriverManager.getConnection(url, user, password);

                                    // 2. Crear la consulta
                                    String sql = "INSERT INTO usuarios (nombre, correo) VALUES (?, ?)";
                                    PreparedStatement stmt = conexion.prepareStatement(sql);
                                    stmt.setString(1, "Ana");
                                    stmt.setString(2, "ana@example.com");

                                    // 3. Ejecutar
                                    int filas = stmt.executeUpdate();
                                    System.out.println("Filas insertadas: " + filas);

                                    // 4. Cerrar conexión
                                    stmt.close();
                                    conexion.close();

                                } catch (SQLException e) {
                                    e.printStackTrace();
                                }
                            }
                        }
