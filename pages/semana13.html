¿Qué aprendí?

Rutas, Controladores y Vistas en Laravel

Laravel utiliza el patrón MVC (Modelo - Vista - Controlador), lo que permite separar la lógica de negocio, la interfaz y la gestión de datos.

alt_route ¿Cómo funciona el flujo básico?

  • El usuario accede a una URL (ruta).
  • Esa ruta llama a un método de un controlador.
  • El controlador devuelve una vista o una respuesta.

select_check_box Ejemplo práctico


                    // web.php (definición de ruta)
                    Route::get('/saludo', [SaludoController::class, 'mostrar']);

                    // SaludoController.php
                    class SaludoController extends Controller {
                        public function mostrar() {
                            return view('saludo', ['nombre' => 'Ana']);
                        }
                    }

                    // saludo.blade.php
                    <h1>Hola, {{ $nombre }}!</h1>

Conexión a Base de Datos y Eloquent ORM

Laravel incluye Eloquent, un ORM (Object-Relational Mapping) que permite trabajar con la base de datos usando clases en lugar de escribir SQL directamente.

database ¿Qué se necesita?

  • Definir la conexión en el archivo .env.
  • Crear un modelo que represente la tabla.
  • Usar métodos como all(), find(), create(), etc.

select_check_box Ejemplo de uso de Eloquent


                    // Modelo Usuario.php
                    class Usuario extends Model {
                        protected $fillable = ['nombre', 'correo'];
                    }

                    // Insertar nuevo usuario
                    Usuario::create([
                        'nombre' => 'Pedro',
                        'correo' => 'pedro@example.com'
                    ]);

                    // Obtener todos los usuarios
                    $usuarios = Usuario::all();

check_circle Ventajas de Eloquent

  1. Facilita las operaciones CRUD sin escribir SQL.
  2. Se integra fácilmente con controladores y rutas.
  3. Permite relaciones entre modelos (uno a muchos, muchos a muchos, etc.).
