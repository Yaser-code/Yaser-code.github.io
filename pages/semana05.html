REACT

React es una librería de JavaScript desarrollada por Meta (Facebook) para construir interfaces de usuario (UI), especialmente en aplicaciones web de una sola página (SPA).

Caracteristicas:

  • Basado en componentes reutilizables.
  • Usa una sintaxis llamada JSX, que mezcla HTML con JavaScript.
  • Muy rápido gracias al DOM virtual.
  • Gran ecosistema y comunidad.
  • Ideal para interfaces dinámicas y aplicaciones modernas.

COMANDO PARA CREAR UN PROYECTO REACT

npx create-react-app nombre-del-proyecto

VITE

Vite (pronunciado "vit", significa "rápido" en francés) es una herramienta de desarrollo frontend que sirve para crear y compilar proyectos web de forma más rápida que Webpack u otras herramientas tradicionales.

Caracteristicas:

  • Desarrollado por el creador de Vue.js.
  • Usa ES Modules nativos del navegador.
  • Carga instantánea del proyecto y cambios en tiempo real (HMR).
  • Compatible con React, Vue, Svelte, TypeScript, etc.
  • Ideal para desarrollos modernos y ligeros.

COMANDO PARA CREAR UN PROYECTO VITE

npm create vite@latest mi-proyecto
