Inversión de Control (IoC) y Dependencias

En Spring, la Inversión de Control (IoC) permite que el framework gestione la creación y conexión de objetos, favoreciendo un código limpio, desacoplado y fácil de mantener.

cable ¿Qué es la inyección de dependencias (DI)?

  • Es la forma en que Spring suministra automáticamente los objetos que una clase necesita.
  • Se puede hacer por constructor o por atributos.
  • Evita crear instancias manualmente con new.

view_comfy_alt Anotaciones comunes

  • @Component: Marca una clase para que Spring la gestione como un bean.
  • @Autowired: Le indica a Spring que debe inyectar una dependencia automáticamente.

select_check_box Ejemplo de IoC + DI


                        @Component
                        public class Motor {
                            public void encender() {
                                System.out.println("Motor encendido");
                            }
                        }

                        @Component
                        public class Auto {
                            private final Motor motor;

                            @Autowired
                            public Auto(Motor motor) {
                                this.motor = motor;
                            }

                            public void arrancar() {
                                motor.encender();
                            }
                        }

Creación de APIs REST con Spring MVC

Spring MVC permite construir APIs REST de forma sencilla, usando anotaciones para mapear peticiones HTTP a métodos en clases controladoras.

api ¿Qué hace un controlador REST?

  • Escucha solicitudes HTTP como GET, POST, PUT, DELETE.
  • Procesa datos y devuelve respuestas en formato JSON.
  • Utiliza anotaciones como @RestController y @RequestMapping.

select_check_box Ejemplo práctico de controlador REST


                        @RestController
                        @RequestMapping("/usuarios")
                        public class UsuarioController {

                            @GetMapping("/{id}")
                            public String obtenerUsuario(@PathVariable int id) {
                                return "Usuario con ID: " + id;
                            }

                            @PostMapping
                            public String crearUsuario(@RequestBody Usuario usuario) {
                                return "Usuario creado: " + usuario.getNombre();
                            }
                        }

check_circle Ventajas de usar Spring MVC

  1. Facilita la creación de APIs RESTful de forma estructurada.
  2. Usa anotaciones intuitivas y legibles.
  3. Permite integrar fácilmente con bases de datos o servicios.
