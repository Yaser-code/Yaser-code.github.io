Uso de HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web.

HTML le dice al navegador qué debe mostrar en una página. Usa etiquetas (tags) para definir diferentes elementos

check Estructura básica:

<nombre_de_etiqueta> Contenido </nombre_de_etiqueta>

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Título de la página</title>
</head>
<body>
<h1>Hola, mundo</h1>
<p>Este es un párrafo.</p>
</body>
</html>

Etiquetas principales:

  • <!DOCTYPE html> : Indica la versión del documento HTML (HTML5).
  • <html> : Inicio del documento HTML.
  • <head> : Sección de configuración (título, estilos, metadatos, etc.).
  • <body> : Es donde va el contenido visible.

Elementos Básicos del encabezado:

  • <meta charset="UTF-8"> : Define el conjunto de caracteres para soportar tildes, ñ, etc.
  • <title> : Define el título que aparece en la pestaña del navegador.

Uso de CSS

CSS significa Cascading Style Sheets (Hojas de Estilo en Cascada).

Es un lenguaje de diseño que se utiliza para dar estilo a las páginas web creadas con HTML.

check ¿Que puede hacer?

  • Cambiar colores, fuentes y tamaños
  • Posicionar elementos en la página
  • Agregar márgenes y bordes
  • Hacer que tu web se vea bien en celulares y computadoras (diseño responsivo)
  • Agregar efectos como animaciones o transiciones

Formas de uso:

Ubicación Código
En linea (Dentro de una etiqueta de html) <h1 style="color: red;">Hola</h1>
Interno (dentro de <style> en el <head>) <head>
<style>
p {color: green;}
</style>
</head>
Externo (en un archivo .css separado) <head>
<link rel="stylesheet" href="estilos.css">
</head>

Selecctores en CSS:

Selector Descripción Ejemplo (como texto)
Etiqueta Selecciona todas las etiquetas de un tipo p { color: blue; }
Clase Selecciona elementos con una clase .miClase { font-size: 18px; }
ID Selecciona un elemento con un ID único #miID { background-color: yellow; }
Universal Selecciona todos los elementos * { margin: 0; }
Descendiente (espacio) Selecciona elementos dentro de otros div p { color: red; }
Hijo directo `>` Selecciona solo los hijos directos ul > li { list-style: none; }
Hermano adyacente `+` Selecciona el siguiente hermano inmediato h1 + p { color: green; }
Hermano general `~` Selecciona todos los hermanos posteriores h1 ~ p { font-style: italic; }
Selector agrupado `,` Selecciona múltiples elementos h1, h2, p { margin: 10px; }

Principales propiedades:

Propiedad Descripción Ejemplo
color Color del texto color: red;
background-color Color de fondo del elemento background-color: yellow;
font-size Tamaño del texto font-size: 16px;
font-family Tipo de fuente (letra) font-family: Arial, sans-serif;
text-align Alineación del texto (left, center, right) text-align: center;
margin Espacio exterior alrededor del elemento margin: 10px;
padding Espacio interior entre el borde y el contenido padding: 5px;
border Borde del elemento border: 1px solid black;
width Ancho del elemento width: 300px;
height Alto del elemento height: 200px;
display Tipo de caja: block, inline, flex, etc. display: flex;
position Ubicación del elemento (relative, absolute, fixed, etc.) position: relative;
overflow Cómo se comporta el contenido desbordado overflow: auto;
z-index Orden de apilamiento de elementos z-index: 10;
